# MysticoN++ C64
MysticoN++ C64 style converted for Notepad++ 7.9.x (thanks to ChatGPT 3.5)

Installation:
- Copy *.xml file to %appdata%\Notepad++\themes 
- How to go to %appdata%: Start -> Run -> type: %appdata%
